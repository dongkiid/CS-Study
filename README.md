## ⏰ 스터디 일정 및 규칙
- 시간: 매주 목요일 10:00 ~ 13:00
- 장소: 오프라인, 신논현역
- PR: 오프라인 발표 이후 자신이 발표한 자료를 업로드
- 벌금: 지각 10분당 ₩500

## 🏋️‍♀️ 스터디 목표
- 컴퓨터 운영체제에 대한 지식을 습득하여 더 나은 개발자로 성장해 나간다.

## 👨🏻‍💻 스터디 진행 방법
- 일주일에 5개 챕터를 공부하고 주제 하나씩 맡아서 약 10분 정도 발표를 진행
- 주제는 사다리타기를 통해 공평하게 분배
